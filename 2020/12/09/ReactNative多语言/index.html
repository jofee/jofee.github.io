ReactNative多语言

发表于 2020-12-09 更新于 2022-01-11

主要使用I8n.js和react-native-localize两个库。
以中英文两种语言为例：

  1. 安装I18n.js,使用命令 yarn add i18n-js或 npm install i18n-js.
  2. 安装 react-native-localize,使用命令 yarn add react-native-localize 或 npm install --save react-native-localize
  3. 创建 i18n 文件夹，在 i18n 文件夹中创建 locales 文件夹，在 locales 文件夹中创建 zh.js 和 en.js 两个多语言文件，写入需要展示的多语言文本，例如：

zh.js

en.js

  1. 在 i18n 文件夹中创建 i18n.js

i18n.js

  1. 使用 i18n,此时是根据系统语言来展示的。
